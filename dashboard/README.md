# Smart Sensor Dashboard

This module represents the operator-facing monitoring surface for the Smart Sensor Network Platform.

## Highlights
- The current MVP dashboard is served from `cloud_pipeline/api/static/dashboard.html`.
- This folder is reserved for the standalone dashboard implementation.
- Focus area: live node status, event stream, alerts, and acknowledgements.
- Designed for operational clarity and fast triage.
- Intended to evolve into a dedicated frontend surface.
